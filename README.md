# Cassiopeia3
Cassiopeia template for Joomla 3.x

**Package**
-Template Cassiopeia for Joomla 3.x
-Plugin Multiplugin for replacing jQuery 3.3.1 & Bootstrap 4.1.3
-Plugin Modules for adding modules for Articles
